如何安装
---
您可以长按全选复制，发送给您习惯的ai，大部分ai都可以为您提供操作帮助
---
第零步：准备工作
由于有道词典笔的特殊性，虽然支持adb，但不是安卓(Android)系统，所以无法使用便利的各种adb工具箱，请准备好以下内容：
adb命令行工具
已经得到密码的有道词典笔
用于连接设备和词典笔的数据线
能够理解命令行英文的头脑或工具
---
第一步：连接设备并认证
adb shell auth success
---
第二步：推送安装包到设备
adb push [电脑上的文件路径] [设备目标路径]
示例：
adb push C:\Users\hdh\Desktop\loli_v1.0.amr /userdisk/loli_v1.0.amr
成功提示：
1 file pushed, 0 skipped. 136.1 MB/s (262295 bytes in 0.002s)
---
第三步：使用 miniapp_cli 安装
adb shell miniapp_cli install [设备上的文件路径]
示例：
adb shell miniapp_cli install /userdisk/loli_v1.0.amr

# 兼容性检测

{: .note }
如果你有不理解的部分，可以长按全选复制，发送给你习惯的 AI，大部分 AI 都可以为你提供操作指导。

通过 adb 执行以下五条指令，将输出结果发到社群中询问：

```bash
uname -m
cat /proc/device-tree/compatible
ls -la /usr/bin/
ls /dev
gst-launch-1.0
```

也可以直接报设备型号在群里问（不一定有结论）。

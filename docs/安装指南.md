# 安装指南

{: .warning }
有道词典笔虽支持 adb，但并非标准 Android 系统，不能使用常规 adb 工具箱，请通过命令行操作。

{: .note }
如果不理解以下内容，可以长按全选复制，发送给你习惯的 AI，大部分 AI 都可以为你提供操作帮助。

---

## 前置条件

- 已安装 adb 命令行工具
- 已获取词典笔的 adb 密码（见[解锁教程](https://github.com/86lbs/ydpen-adb-unlock)）
- 数据线连接电脑与词典笔

---

## 第一步：连接并认证

```bash
adb shell auth success
```

## 第二步：推送安装包到设备

```bash
adb push <本地文件路径> /userdisk/<文件名>.amr
```

示例：

```bash
adb push C:\Users\你的用户名\Desktop\loli_v1.5.amr /userdisk/loli_v1.5.amr
```

## 第三步：安装

```bash
adb shell miniapp_cli install /userdisk/<文件名>.amr
```

示例：

```bash
adb shell miniapp_cli install /userdisk/loli_v1.5.amr
```
